Sobre o Projeto

O Projeto JSPT - Javascript Portugol tem por objetivo fornecer um ambiente para o aprendizado de linguagens de programação e algoritmos utilizando a linguagem Portugol.

Portugol

O Portugol é uma linguagem de pseudocódigo totalmente em Português muito utilizada nos cursos iniciais de programação. O JSPT utiliza um dialeto de Portugol inspirado na linguagem G-Portugol.

Exemplo de programa: 
algoritmo OlaMundo;
início
   escreva("Olá, Mundo!");
